Lattice Representation and Geometry Generation

Lattice graph Strut / joint cleanup Strut / joint SDF Implicit field Surface geometry
  • The system input is a generalized periodic lattice graph.
  • Periodic edges crossing unit-cell boundaries are represented in the same graph workflow.
  • Equivalent graph inputs are cleaned into a consistent strut/joint representation before implicit conversion.
  • Struts and joints are defined using signed-distance / implicit fields.
  • Radius and smoothness are treated as continuous design variables.
  • Eight well-known lattices are used as validation/comparison cases.
Current benchmark set: SC, BCC, FCC, diamond, Kelvin, octahedron, octet, and rhombic.
