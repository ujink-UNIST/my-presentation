Dataset and Structural Representation

28-dimensional binary topology vector

  • Each binary variable represents the existence of a candidate structural member.
  • Mechanical response is a quasi-static compression stress–strain curve.
  • The dataset contains more than 20,000 FE-simulated 3D lattice topologies.
Connection to this project: the dataset pipeline is very similar: define lattice topology → generate FE model → run simulation → store response data. The key difference is that this work uses graph/implicit geometry and targets Cᴴ plus modal outputs.
Fixed candidate members make the paper's topology representation compact, but the design space is bounded by the predefined bar set.
