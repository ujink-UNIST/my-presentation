Research Objective and Scope

Current objective

Automated lattice evaluator

Convert a periodic lattice graph into implicit geometry and a solid mesh, then extract stiffness, density, and low-order modal outputs.

Long-term objective

Active-learning-compatible FEM backend

Select informative lattice candidates under limited simulation budget and build reliable data for downstream AI design.

Evaluator boundary

Periodic graph
Implicit geometry
Solid FEM
\(\mathbf{C}^{H},\rho_{rel}\)
\(f_1\sim f_4\)
mode shapes

Static outputs

  • Homogenized stiffness matrix Cᴴ
  • Relative density ρrel
  • Consistent result schema for AI-ready datasets

Modal outputs

  • 1st–4th lattice eigenfrequencies
  • 1st–4th lattice mode shapes
  • Reusable inputs for later seat-frame-level design
