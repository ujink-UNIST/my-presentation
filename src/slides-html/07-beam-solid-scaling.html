Beam and Solid Models Predict Different Scaling

Key message

Beam and solid homogenization predict different topology-dependent scaling.

  • Left plot keeps the full measured range, including VF ≥ 1.
  • The shaded region marks VF ≥ 1, where beam volume fraction can become nonphysical due to overlap counting.
  • Color indicates unit-cell type; line style and marker type indicate beam or solid model.
  • The exponent plot uses VF < 1 for fitting.

Solid FEM is retained as the high-fidelity evaluator.
