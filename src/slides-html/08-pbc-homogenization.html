Unit-Cell Homogenization with PBC

Why periodic boundary conditions?

PBC is selected because it is consistent with an infinitely repeating lattice assumption.

\(\mathbf{u}^{+}-\mathbf{u}^{-}=\bar{\boldsymbol{\varepsilon}}(\mathbf{x}^{+}-\mathbf{x}^{-})\)
  • Apply independent macroscopic strain load cases.
  • Constrain paired nodes on opposite faces.
  • Compute volume-averaged stress or reaction-force response.
  • Assemble the homogenized stiffness matrix \(\mathbf{C}^{H}\).
  • Store relative density and run a separate lattice modal analysis for \(f_1\sim f_4\) and mode shapes.
